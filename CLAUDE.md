## What This Is

ARIA-OS — a multi-domain AI-driven autonomous engineering pipeline.
Natural language → physics-constrained CAD → validated STEP/STL → DFM → Quote → CAM.

The pipeline is fully agentic: local Ollama LLMs handle orchestration,
cloud LLMs (Anthropic/Gemini) handle complex code generation.
